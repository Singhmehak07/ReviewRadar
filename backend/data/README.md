# ReviewRadar Raw Dataset

This directory contains the raw untransformed training dataset.

## Dataset Information
*   **Dataset Name**: Fake Reviews Dataset
*   **Original Source**: Liyuan Li et al.
*   **License**: Open-data/Creative Commons attribution.
*   **Purpose**: Used to train the TF-IDF Vectorizer and Logistic Regression models.
*   **Git Exclusions**: The full dataset is backed up outside the repository and ignored locally to optimize Git repository size.
